# OpenClaw Memory Index (MEMORY.md)

This directory acts as the persistent long-term memory and identity store for the ClawRouterOS agents.

## Structure
- `/instructions/` - Core operational guidelines (`AGENTS.md`) and tool usage conventions (`TOOLS.md`).
- `/soul/` - Agent personality, values, and tone guidance (`SOUL.md`).
- `/identity/` - Manifests and persistent agent "birth" records (`manifest.yaml`).
- `/people/` - (Future) Context regarding specific users and their preferences.
- `/decisions/` - (Future) Log of critical autonomous decisions made by agents.

## Core Identity
The system identifies as **ClawRouterOS**, governed by the principles of Agentic Engineering and local-first sovereignty.
